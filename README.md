## 🧠 Attack Methodology

Every room I complete follows this systematic approach:

```
1. RECONNAISSANCE
   └─ Service enumeration, version detection
   
2. VULNERABILITY SCANNING
   └─ Identify exploitable weaknesses
   
3. EXPLOITATION
   └─ Gain initial foothold with proof-of-concept
   
4. PRIVILEGE ESCALATION
   └─ Escalate from user to system level
   
5. POST-EXPLOITATION
   └─ Credential harvesting, lateral movement, persistence
   
6. DOCUMENTATION
   └─ Complete writeup with reasoning & lessons learned
```
